# Bioregenerative Life-support System for Interplanetary Spacecraft (BLiSIS)

> Conceptual engineering project of an autonomous bioregenerative life support system for long-duration interplanetary missions.

---

## Overview

This project presents a conceptual design of a **Bioregenerative Life Support System (BLSS)** intended to support a crew of **10 astronauts** during an approximately **1000-day Mars mission**.

The system is based on a closed-loop resource recycling approach, minimizing dependency on Earth resupply. All modules are designed to fit within the pressurized volume of **SpaceX Starship**.

# Main Modules

## Photobioreactor

The photobioreactor uses **Chlorella vulgaris** for atmospheric regeneration and biomass production.

**Functions:**
- oxygen production  
- carbon dioxide absorption  
- protein-rich biomass generation  

<p align="center">
  <img src="images/photobioreactor.png" width="650">
</p>

---

## Insect Module

Organic waste processing system based on **black soldier fly larvae (Hermetia illucens)**.

**Functions:**
- organic waste recycling  
- protein and fat production  
- nutrient recovery loop

## Aeroponic Greenhouse

A seven-level aeroponic greenhouse for food production in space conditions.

**Primary crops:**
- sweet potato  
- potato  

**Additional crops:**
- wheat  
- amaranth  
- leafy greens  
- herbs  
- vegetables
